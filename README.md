# Гончарная мастерская «Глина» — MVP

Клиент-серверное веб-приложение для записи на мастер-классы гончарной мастерской.
Тестовое задание (AI-driven development): полный цикл от требований до кода и тестов.

## Структура

```
docs/       отчёты по этапам (аналитика, проектирование, разработка, тестирование)
backend/    REST API на Python/Flask (in-memory данные)
frontend/   3 экрана на HTML/CSS/JS без фреймворков
```

## Запуск

**Бэкенд:**
```bash
cd backend
pip install -r requirements.txt
python api.py          # http://127.0.0.1:5000
```

**Фронтенд:** открыть `frontend/index.html` в браузере
(или `python -m http.server 5500` из папки `frontend`).

## Экраны

* `index.html` — расписание занятий (7 дней, R-027, empty state).
* `details.html` — карточка занятия и форма записи.
* `success.html` — подтверждение брони.

## Бизнес-правила

* Горизонт расписания — 7 дней (R-027).
* Вместимость: гончарный круг — до 6 чел., лепка — до 10 чел.
* Прокат инструментов и фартука — без размеров.
* Атомарная проверка мест, 409 при отсутствии (R-004).
* Запрет записи на слоты `CANCELLED_BY_STUDIO` (R-008).

# Гончарная мастерская «Глина» — MVP

Клиент-серверное веб-приложение для записи на мастер-классы гончарной мастерской.
Тестовое задание (AI-driven development): полный цикл от требований до кода и тестов.

## Структура

```
docs/       бриф заказчика + отчёты по этапам (аналитика, проектирование,
            разработка, тестирование, доработки)
backend/    REST API на Python/Flask (in-memory данные)
frontend/   7 экранов на HTML/CSS/JS без фреймворков
```

## Запуск

**Бэкенд:**
```bash
cd backend
pip install -r requirements.txt
python api.py          # http://127.0.0.1:5000
```

**Фронтенд:** открыть `frontend/index.html` в браузере
(или `python -m http.server 5500` из папки `frontend`).

## Экраны

* `index.html` — расписание занятий: группировка по дням, фильтры по программе,
  горизонт 7/14 дней (R-027), empty state.
* `details.html` — карточка занятия, прокат, форма записи с итоговой ценой.
* `success.html` — подтверждение брони.
* `masters.html` — команда мастеров с рейтингами.
* `bookings.html` — «Мои записи»: поиск по телефону, отмена брони.
* `login.html` — вход (демо-аккаунты клиента и владельца).
* `admin.html` — панель владельца: всё расписание, отмена/восстановление
  занятий, ручная правка числа мест.

## Роли и вход

| Логин | Пароль | Роль |
|-------|--------|------|
| `admin` | `admin123` | Владелец (админка) |
| `client` | `client123` | Клиент (демо-записи) |

## Бизнес-правила

* Горизонт расписания — 7 дней (R-027), расширение через фильтр дат.
* Вместимость: гончарный круг — до 6 чел., лепка — до 10 чел.
* Прокат инструментов и фартука — без размеров.
* Атомарная проверка мест, 409 при отсутствии (R-004).
* Запрет записи на слоты `CANCELLED_BY_STUDIO` (R-008).
* Отмена брони клиентом — не позднее чем за 2 часа до начала, место освобождается.

## Демо-данные

Для экрана «Мои записи» посеяны брони на телефон `+7 900 000-00-00`.

# Задача 4. Тестирование и Сводный верификационный отчет

* Тестирование проводилось на строгое соответствие правилам **R-027** (горизонт вывода), **R-004** (запрет овербукинга при параллельных запросах), **R-008** (запрет записи на отмененные слоты) и **R-cancel** (блокировка отмены менее чем за 2 часа).

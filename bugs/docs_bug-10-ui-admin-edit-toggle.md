# Баг 10. Бесполезная кнопка-переключатель формы правки слота в админке

## 1. Симптом бага
Кнопка «Перенести · сменить мастера» не скрывала и не показывала форму правки — инлайн-форма была видна постоянно.

## 2. Ссылка на бизнес-требование
* **Требование:** UI/UX Панели

## 3. Причина возникновения и Реализация исправления
* **Причина:** JS переключал HTML-атрибут `hidden`, но CSS-правило `.admin-row__edit { display: flex }` имело более высокий приоритет, полностью подавляя браузерный стиль скрытия элементов.
* **Фикс:** Убран нерабочий переключатель и сама кнопка «Отмена». Инлайн-форма правки зафиксирована как постоянный элемент под строкой слота.

## 4. Отправленные промпты
> Кнопка переключения формы правки в админке ничего не делает, так как display: flex в CSS перебивает атрибут hidden. Убери кнопку и зафиксируй форму.

## 5. Результат ручной проверки и Commit
* **Статус:** Исправлено. Поведение системы верифицировано вручную, баг полностью устранен и не воспроизводится.
* **Commit:** `fix(cors): разрешить Authorization; убрать нерабочую кнопку формы правки`

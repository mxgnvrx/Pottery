# Баг 08. Разрушение верстки блока сводки админки на десктопе

## 1. Симптом бага
При разрешении экрана более 900px блок со статистикой админки разъезжался и выстраивался в вертикальную колонку-карточку.

## 2. Ссылка на бизнес-требование
* **Требование:** Вёрстка

## 3. Причина возникновения и Реализация исправления
* **Причина:** Блок ошибочно унаследовал CSS-класс `.hero__stats`, имеющий жесткие десктопные правила flex-колонки внутри главного баннера.
* **Фикс:** Для блока сводки выделен собственный независимый класс `.stats-row`, гарантирующий стабильный горизонтальный ряд элементов.

## 4. Отправленные промпты
> Сводка админки на десктопе ломается и встает вертикально. Это из-за наследования стилей hero. Выдели для нее отдельный класс CSS.

* **Статус:** Исправлено. Поведение системы верифицировано вручную, баг полностью устранен и не воспроизводится.

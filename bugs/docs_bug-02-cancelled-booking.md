# Баг 02. Запись на отмененное мастерской занятие

## 1. Симптом бага
Бронь слота со статусом `CANCELLED_BY_STUDIO` успешно обрабатывалась сервером с кодом 201.

## 2. Ссылка на бизнес-требование
* **Требование:** R-008

## 3. Причина возникновения и Реализация исправления
* **Причина:** В критической секции `create_booking` полностью отсутствовал контроль статуса самого занятия.
* **Фикс:** Добавлена проверка: `if slot['status'] == STATUS_CANCELLED_BY_STUDIO: return jsonify(...), 409`.

## 4. Отправленные промпты
> Прогони TC-11 через Flask test client. Почему бронь отмененного слота возвращает 201 вместо 409? Добавь проверку статуса в бэкенд.

## 5. Результат ручной проверки и Commit
* **Статус:** Исправлено. Поведение системы верифицировано вручную, баг полностью устранен и не воспроизводится.
* **Commit:** `fix: запрет брони заполненных и отменённых слотов (R-004, R-008) + UI`

# Баг 05. Фильтр «14 дней» не выводил новые занятия

## 1. Симптом бага
При нажатии кнопки «Показать 14 дней» список занятий визуально не расширялся.

## 2. Ссылка на бизнес-требование
* **Требование:** R-027

## 3. Причина возникновения и Реализация исправления
* **Причина:** В демо-данных `seed_data()` за пределами первой недели генерировался всего один тестовый слот, что делало переключение незаметным.
* **Фикс:** Расширена генерация начального наполнения расписания на полноценные 14 дней вперед с плотным распределением по 3 сеанса в сутки.

## 4. Отправленные промпты
> Кнопка 'Показать 14 дней' работает, но список не меняется. Проверь seed-данные на бэкенде, добавь слоты на дальние даты.

## 5. Результат ручной проверки и Commit
* **Статус:** Исправлено. Поведение системы верифицировано вручную, баг полностью устранен и не воспроизводится.
* **Commit:** `fix: дубль даты в «Моих записях» + слоты за горизонтом 7 дней`
